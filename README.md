## DSE Math Algorithm Black Box

---

### 📊 產品簡介

這不是一本數學筆記，而是一份針對考評局出題邏輯的破解方案。

基於對 2012-2025 年共 14 年 DSE 數學 Paper 2 選擇題的深度數據分析，揭示了考評局答案編排的隱藏規律，並提供了一套完整的時間管理和答題策略。

### ✨ 核心內容

#### Chapter 1: 答案分佈規律
- 甲部「7-8-7-8」均衡鐵律
- 乙部「3-4-4-4」固定模型
- 全卷「11-11-11-12」黃金比率
- 預測準確率：92.8%

#### Chapter 2: 時間管理策略
- 75分鐘精準分配方案
- 策略跳題技巧（Q16/Q17）
- 倒序做題法（Q45→Q42）
- 中場答案分佈檢查

#### Chapter 3: 終極猜題技巧
- 基於統計學的智能猜題法
- 比隨機猜測提升 200%+ 正確率
- 最後 5 分鐘救命策略

### 📈 數據支持

- **分析時間跨度**：2012-2025（14年）
- **總樣本題數**：630題
- **規律符合率**：92.8%
- **統計方法**：Python pandas + numpy
- **數據來源**：考評局官方公佈答案
